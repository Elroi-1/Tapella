## **Tapela — Digital Business Card & Advertising Platform**

Tapela is a Flutter mobile application where business owners create digital business cards to advertise their services, and users can search by category and location to find nearby businesses. The system includes authentication, runs with a local REST API and local database, and supports CRUD operations for managing business cards and user favorites, all fully offline.
